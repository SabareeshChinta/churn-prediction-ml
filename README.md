# 📊 Customer Churn Prediction System

An end-to-end Machine Learning project that predicts whether a customer is likely to churn using Logistic Regression.

---

## 🚀 Features

* Real-world dataset (Telco Customer Churn)
* Data cleaning & preprocessing
* Logistic Regression model
* Model evaluation (accuracy, confusion matrix, classification report)
* Deployed using Streamlit

---

## 🧠 Problem Statement

Customer churn is a major issue for businesses. This project aims to predict whether a customer will leave a service based on their usage and billing information.

---

## ⚙️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

---

## 📊 Model Performance

* Accuracy: ~0.80
* Evaluated using confusion matrix and classification report
